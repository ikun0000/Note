#### Github搜索语法

查找包含关键词的仓库名

```
in:name 关键词
```

 查找描述的内容 

```
in:descripton 关键词
```

查找README文件包含特定关键词 

```
in:readme 关键词
```

查找star数大于x的仓库

```
stars:>x
```

按大小查找仓库（单位是`KB`）

```
size:>=x
```

查找最晚一次push的时间

```
pushed:>YYYY-MM-DD
```

查找创建时间在YYY-MM-DD之后的仓库

```
created:>YYYY-MM-DD
```

查找指定协议的仓库

```
license:apache-2.0
```

按语言查找仓库

```
language:java
```

查找某个用户的仓库

```
user:username
```
