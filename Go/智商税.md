# 科学判断智商税

```flow
花钱=>operation: 花钱
达到目的了吗=>condition: 达到目的了吗
成本失控了吗=>condition: 成本失控了吗
目的和成本有因果吗=>condition: 目的和成本有因果吗
爽吗=>condition: 爽吗
钱没了=>operation: 钱没了
智商税=>operation: 智商税
花钱->达到目的了吗
达到目的了吗(yes)->成本失控了吗
达到目的了吗(no)->智商税
成本失控了吗(yes)->智商税
成本失控了吗(no)->目的和成本有因果吗
目的和成本有因果吗(yes)->爽吗
目的和成本有因果吗(no)->智商税
爽吗(yes)->钱没了
爽吗(no)->智商税
```
